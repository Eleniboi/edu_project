# ➤ Exercise 3: The Fairness Contract
**Objective:** Define your personal rules for ethical AI use in learning.
Create a markdown file fairness_contract.md with three sections:

### *section 1:* I will use AI when:

❖ I have tried to understand what the problem is at first and how to solve it\
❖ I seek for more deeper understanding\
❖ I have tried possible solution by myself 

### *section2:*  I Will NOT use AI when:
❖ I haven't know what the problem is \
❖ They are other possible ways to solve the problem without AI\
❖  Am still learning Basic concept 
### *section3:* I know I Am using AI fairly when:
❖ I can explain each line of my code without AI's help\
❖ I could solve similar problem on my own \
❖ I feel more bold with my own understanding
