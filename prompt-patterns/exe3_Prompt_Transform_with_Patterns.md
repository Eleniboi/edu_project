## Objective: Strengthen weak prompts using patterns.

### 1. Take a vague or under-specified prompt (e.g., "Write about climate change.").

**under-specified prompt** 
"write about programming"

### 2. Transform it using instructional or comparative patterns:

- **COMPARATIVE:** "In 100-word explain programing and 5 programming speciallization in bullet."

### 3. Document the improvement in clarity and usefulness of the outputs.

**under-specified prompt output**

Programming is essentially the art of teaching a rock how to think. We take silicon, apply some electricity, and then spend hours arguing with a text editor about where a semicolon went.

At its core, programming is structured problem-solving. It’s the bridge between a human’s messy, creative idea and a computer’s rigid, logical execution.

- **COMPARATIVE output:**

Programming is the process of creating a precise set of instructions that a computer follows to perform tasks. It’s a blend of logic and creativity; you identify a problem, break it into tiny, actionable steps, and translate those steps into a language the machine understands. While the syntax varies between languages like Python or C++, the core goal remains the same: automation and problem-solving. It is the invisible engine behind everything from your microwave's timer to the complex neural networks powering modern AI.

Here are five key programming specializations:
```
   - Front-End Development: Crafting the visual elements and user interfaces of websites.

    - Back-End Development: Managing servers, databases, and the internal logic "under the hood."

    - Data Science: Using code to analyze massive datasets and build predictive models.

    - Mobile Development: Creating specialized applications for iOS and Android devices.

    - DevOps: Automating the bridge between software development and IT operations for faster deployment.
```
### DOCUMENTATION:

**clarity**
1. With the first prompt AI doesn't know if you want a history, a tutorial, or a career guide on a choosen topic you mentioned , so there is no clarity in the vague prompt, while the comparative prompt give the AI a structured blue print, an exact word length to use, here the clarity is high.

**usefulness**

Using vague prompt,you have to "dig" for the information you actually need, here the AI model assume what you want, it might output thing that you don't need, while the comparative prompt produce a quick executive summary ready for a presentation or note following the prompt structure.
