# prompt-patterns
# Overview

    Reusable prompt structures for different tasks

    Instructional, comparative, QA, and data extraction patterns

## Learning Objectives

    Learn key prompt design patterns

    Transform a single use-case into a reusable prompt template

    Improve weak prompts by applying structured patterns

    
**Reflection:**
The whole project was structured in a way that it teaches you how to structure your prompt by giving the AI Model a blueprint-like instructions, on how to give response in a specific format , this avoid or prevent halucination by the AI Model.
