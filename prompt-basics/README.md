# Exercise 1: Simple Prompt Writing and Output Observation

## 📖 Objective

The objective of this exercise is to practice writing different types of prompts and observe how an AI model responds to each prompt style.

## 🧪 Prompts Used

### 1. Direct Instruction Prompt

**My Prompt:**
"List three common fruits."

**AI Response:**

* Apple
* Banana
* Orange

This response was short, direct, and precise because the prompt was a clear instruction.

### 2. Question-Style Prompt

**My Prompt:**
"What are the benefits of orange for a sick person?"

**AI Response (Summary):**
The AI provided a detailed explanation including the benefits of vitamin C, hydration, antioxidants, and immune system support for a sick person.

## 🔍 Comparison of Responses

The direct instruction prompt produced a brief and structured answer that matched the exact request.
The question-style prompt produced a more detailed, flexible, and explanatory response with additional context and health-related information.

## 🧠 Reflection

This exercise helped me understand that the way a prompt is written affects the type of response an AI model gives. Direct prompts lead to short and specific answers, while question-style prompts encourage more detailed and explanatory responses.
I learned that clear and well-structured prompts improve the quality of AI-generated outputs.

## 📚 Conclusion

Through this exercise, I gained a foundational understanding of prompt writing and how different prompt styles influence AI behavior and output depth.
