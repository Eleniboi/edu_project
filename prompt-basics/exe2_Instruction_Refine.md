## 1. Start with a vague request like: “Tell me about AI.”

**2. Refine it step by step into a clear instruction e.g.,**

*“Explain AI in simple terms for beginners"*

*“Summarize the main AI techniques in 5 bullet points.”*


### personal refine instruction

- "Explain programming in simple terms for beginners."
- "Summarize the main programming techniques in 5 bullet points."
### Output 

A computer is incredibly fast, but it isn't actually "smart." It doesn't know what you want it to do until you give it a specific set of instructions. Programming is simply the act of creating those instructions.

Variables & Data Storage: Saving information (like names, numbers, or text) so the program can use and update it later.

Input & Output: Taking data from the user (input) and displaying results or messages back (output).

Conditionals (Decision Making): Using logic like “if/else” to make the program choose different actions based on situations.

Loops (Repetition): Repeating tasks automatically instead of writing the same instruction many times.

Functions (Reusable Code): Creating small blocks of code that perform a task and can be reused to keep programs organized and efficient.
